# 葛洲坝船闸导航系统项目文档

## 项目概述

葛洲坝船闸导航系统是一个基于Go语言开发的智能船闸管理系统，主要用于葛洲坝船闸的船舶导航、状态监控、速度测量和禁停线监测等功能。
