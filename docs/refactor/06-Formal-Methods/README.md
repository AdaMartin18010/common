- 结合神经网络与符号推理，实现自动定理证明、模型检验、程序综合等创新应用。
- 典型工具：CoqGym、HOList、Certora Prover、Neural TLA+、GPT-Prove 等。
- 未来展望：大规模自动化验证、AI辅助形式化建模、可解释性增强。
